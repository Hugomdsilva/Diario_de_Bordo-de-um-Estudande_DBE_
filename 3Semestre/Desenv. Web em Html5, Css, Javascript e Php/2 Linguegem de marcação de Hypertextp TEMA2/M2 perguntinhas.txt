oque sao as funções estruturais, textuais e semanticas das tags?
    R : tags extruturais sao obrigatorias a uma pagina sao elas as textuais e as semanticas
    Tags Textuais :responsaveis por Organizar O Conteudo Da Pagina ou seja textos midias e links algums exemplos
    sao <p>, <h1> ... <h6>, <img> e <a>.
    Tags Semanticas : responsaveis por Organizar A Estrutura De Conteudo De Uma Pagina foram inseridas a
    partir do HTML5 um exemplo seria usalar com a estrutura basica de uma pagina ai temos as tags <header>
    <nav> <main> <aside> <footer>.

oque sao tags obsoletas? porque elas se tornaram obsoletas? 
     R: a cada nova versão do html são criadas novas tags e as antigas descontínuadas que são chamadas de obsoletas. muitas vezes o navegador tem suporte a essas tags porém não é recomendado ultilizar.
porque se ela se tornou obsoleta foi criada uma nova tag com a mesma função ou até melhor.
e também usar tags obsoletas colocam o  código em risco já que a qualquer momento o navegador pode remover o suporte aquela tag
