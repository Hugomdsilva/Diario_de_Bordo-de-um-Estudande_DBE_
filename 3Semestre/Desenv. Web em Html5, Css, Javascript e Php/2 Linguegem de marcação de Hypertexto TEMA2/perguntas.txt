oque é o hytime?
	R:O hytime é um padrao criado pela ISO para possibilitar que hyperdocumentos e hypermidia pudessem
	ser descritos em função de sua estrutura e conteudo ou seja humanizalos e consequentemente utilizados por quaisquer
	sistemas conformante

o dtd padroniza o html em diferentes navegadores?
	R:Sim o DTD define os atributos e requisitos de uma linguegem ou seja ele padroniza quaisquer uso de linguem
	para que haja um padrao por exemplo o html tem seu DTD o xml tem um diferente 

qual a diferença de um DTD para um doctype?
	R:Um DTD nao é igual a um doctype no caso o DTD existe em diversas linguagens ou em suas variaçoes ele define
	atributos e requisitos para essa linguem em questao já o DOCTYPE é propio do html ele serve para juntar as tags 
	que podem ser usadas e as marcações que vao ser carregadas no site

    Tags Textuais :responsaveis por Organizar O Conteudo Da Pagina ou seja textos midias e links algums exemplos
    sao <p>, <h1> ... <h6>, <img> e <a>.
    Tags Semanticas : responsaveis por Organizar A Estrutura De Conteudo De Uma Pagina foram inseridas a
    partir do HTML5 um exemplo seria usar com a estrutura basica de uma pagina ai temos as tags <header>
    <nav> <main> <aside> <footer>.

oque sao tags obsoletas? porque elas se tornaram obsoletas? 
     R: a cada nova versão do html são criadas novas tags e as antigas descontínuadas que são chamadas de obsoletas. muitas vezes o navegador tem suporte a essas tags porém não é recomendado ultilizar.
porque se ela se tornou obsoleta foi criada uma nova tag com a mesma função ou até melhor.
e também usar tags obsoletas colocam o  código em risco já que a qualquer momento o navegador pode remover o suporte aquela tag

Como que se usa as tags de listas e um exemplo de um codigo que usa
essas tags? 
	R: as tags de listas são usadas com uma tag pai e as tags filhas como
se a tag pai identifica uma lista as tags filhas os itens dessa lista um exemplo seria:
<ol> <li> item lista nao ordenada </li> </ol>
<ul> <li> item lista ordenada </li> </ul>
<dl><dt><dd> item lista definiçao</dd></dt></dl>
